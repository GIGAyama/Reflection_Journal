# 📔 ふりかえりジャーナル (Reflection Journal)

**「ふりかえりジャーナル」**は、毎日の5分間で児童のメタ認知を育み、先生との深い信頼関係を築くための次世代教育Webアプリケーションです。

従来の「紙のノート」や「単なるデジタルフォーム」を超え、児童の「心のバイタルサイン」を可視化し、AIが先生のフィードバックを支援することで、学校を**「データに基づき、一人ひとりの心に寄り添う場所」**へとアップデートします。

## ✨ コンセプト：単なる日記から「革命的なインフラ」へ

*   **心のバイタルサイン（エモーショナル・ウェザーマップ）**: 毎日の記述から児童の感情の変化を14日間のヒートマップとして可視化。小さなSOSを「起きる前に」察知します。
    
*   **先生と子供の交換日記**: 評価や採点ではなく、「対話」を目的としたUI設計。先生からの温かいおへんじ（通知）が児童の心に直接届きます。
    
*   **究極のZero-Config（環境構築不要）**: `Node.js`や`npm`は一切不要。2つのファイル（`コード.gs`と`index.html`）をGoogle Apps Scriptにコピペするだけで、スプレッドシートやフォルダが自動で全構築されます。
    

## 🚀 主な機能

### 🎒 児童向け機能（Student UI）

*   **魔法のヒント＆テンプレート**: 「何を書けばいいかわからない」をゼロにする、タップで入力できるヒント辞書（YWT、KPTなど）。
    
*   **過去の自分との対話**: 1ヶ月前の自分のジャーナルがランダムで表示され、今の自分からメッセージを送るリフレクション機能。
    
*   **完全なルビ（ふりがな）対応**: すべてのボタンや文章に美しいルビを完備。低学年でも一人で操作可能。
    
*   **おへんじ通知＆達成感の演出**: 先生からおへんじが届くとポップアップでお知らせ。提出時には画面いっぱいに紙吹雪が舞います。
    
*   **安心の自動保存**: 入力途中に誤って画面を閉じても、入力内容はブラウザに自動で保存されます。
    

### 👩‍🏫 教員向け機能（Teacher UI）

*   **ダッシュボード＆心のバイタル**: 今日の提出率（ドーナツチャート）や、クラスの感情の波（ヒートマップ）を一目で把握。
    
*   **AI要注目アラート**: 「ネガティブな感情が続いている」「急激に文字数が減った」児童をAIが自動ピックアップ。
    
*   **直感的な名簿エディタ**: アプリ画面上から直接、児童の追加・編集・削除が可能。
    
*   **Gemini AI フィードバック支援**: 未返却のジャーナルに対して、ハイライト（文章の抽出）と温かいコメント案をワンクリックで一括生成。
    
*   **一括エクスポート**: 学期末の記録用に、指定した期間・児童のデータを美しいPDFやCSVとして出力可能。
    

## 🛠️ 導入方法（5分で完了！）

プログラミングの知識は一切不要です。以下の手順に従って設定してください。

### 1. プロジェクトの作成

1.  Googleドライブを開き、「新規」>「その他」>「Google Apps Script」を選択。
    
2.  プロジェクト名を「ふりかえりジャーナル」等に変更します。
    

### 2. コードの配置

1.  最初からある `コード.gs` の中身をすべて消し、本リポジトリの `コード.gs` を貼り付けます。
    
2.  左側の「ファイル」の横にある `+` ボタンを押し、「HTML」を選択。ファイル名を `index` にします。
    
3.  `index.html` の中身をすべて消し、本リポジトリの `index.html` を貼り付けます。
    
4.  💾 保存ボタンを押します。
    

### 3. APIキーの設定（AI機能を利用する場合）

1.  左側の ⚙️ 歯車アイコン（プロジェクトの設定）を開きます。
    
2.  一番下の「スクリプト プロパティを追加」をクリック。
    
3.  プロパティに `GEMINI_API_KEY`、値に取得したGeminiのAPIキーを入力して保存します。
    

### 4. アプリの公開（デプロイ）

1.  画面右上の青い「デプロイ」>「新しいデプロイ」をクリック。
    
2.  種類の選択（歯車アイコン）から「ウェブアプリ」を選択。
    
3.  以下の設定でデプロイします：
    
    *   次のユーザーとして実行：**自分**
        
    *   アクセスできるユーザー：**全員**（※学校ドメイン内に限定する場合はドメインを選択）
        
4.  初回のみ「アクセスを承認」画面が出ます。
    
    *   ご自身のアカウントを選択 → 左下の「詳細」をクリック → 「ふりかえりジャーナル（安全ではないページ）に移動」をクリック → 「許可」をクリック。
        
5.  表示された**ウェブアプリのURL**にアクセスすれば完了です！
    

> **💡 Zero-Config について:** 初回アクセス時に、データ保存用の「スプレッドシート」と画像保存用の「Google Driveフォルダ」があなたのGoogleドライブのルート（マイドライブ）に自動的に作成されます。

## 📱 技術スタック

*   **Backend**: Google Apps Script (GAS)
    
*   **Frontend**: React 18, Babel (Standalone)
    
*   **Styling**: Tailwind CSS (CDN)
    
*   **AI Integration**: Google Gemini API (gemini-2.5-flash)
    
*   **Storage**: Google Spreadsheet (Database) / Google Drive (Images/PDFs)
